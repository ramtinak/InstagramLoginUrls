### InstagramLoginUrls

All login uris from scratch.

Purpose: This is educational, since most of current libraries doesn't work with bloks including my library [InstagramApiSharp](https://github.com/ramtinak/InstagramApiSharp), I decided to create this, although the [private version of InstagramApiSharp](https://github.com/ramtinak/InstagramApiSharp/issues/243) works with bloks version.

I've wrote a tiny example for this in ThreadsApiSharp's repo a long time ago> https://github.com/ramtinak/ThreadsApiSharp/issues/1. by the way `Threads APIs` added to [private version](https://github.com/ramtinak/InstagramApiSharp/issues/243)'s library.
